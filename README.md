# Test For Expounder

This repository contains test projects for Expounder.

## Installation

### Solana CLI Tools

Install the Solana CLI tools using following commands:

```bash
curl --proto '=https' --tlsv1.2 -sSfL https://solana-install.solana.workers.dev | bash
sh -c "$(curl -sSfL https://release.anza.xyz/v3.1.6/install)"
```
